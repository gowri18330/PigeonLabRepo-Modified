
Prerequisites:
=============

1. Install Java jdk-1.8 or higher in machin.
2. Install Eclipse IDE , and add TestNG, Maven Plugins.

 Project clone : clone the pigeon hole project from below git url
=============

 url: https://github.com/gowri18330/PigeonLabRepo-Modified.git
 
 
Run Project:
============

To run Test suite: 
----------------

Steps:
 1.import cloned project to eclipse
 2. go to tesng.xml file-> right click on testng.xml -> run as -> TestNG suite
 
 To run individual Test script:
 -----------------------------
 
 Steps:
 1.import cloned project to eclipse
 2.go to src/test/java ->com.PigeonHole.RegressionScripts package --> select scenario Test class --> right click -> run as -> click TestNg test 
 
 
 Results:
 =======
  go to ExtentReopts folder --> open PigeonExtentReport.html file
