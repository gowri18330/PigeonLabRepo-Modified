
Prerequisites:
=============

1. Install Java jdk-1.8 or higher in machin.
2. Install Eclipse IDE , and add TestNG, Maven Plugins.

Git Clone :  clone the pigeon hole project from below git url
==========

 url: https://github.com/gowri18330/PigeonLabRepo-Modified.git
 
 
Run Project:
============

To run suite: 
-----------

Steps:
 1.import cloned project to eclipse
 2. go to tesng.xml-> right click on testng.xml -> run as -> TestNG suite
 
 To run individual script:
 ------------------------
 
 Steps:
 1.import cloned project to eclipse
 2.go to com.PigeonHole.RegressionScripts package --> select scenario Test class --> right click -> run as -> click TestNg test
